# VideoChat

[WebRTC](https://developer.mozilla.org/ja/docs/Web/API/WebRTC_API)を利用したビデオチャットアプリケーション

## 構成

- ```offer.html``` 送信側
- ```receiver.html``` 受信側

## 技術スタック

- HTML/CSS
- JavaScript
- PHP
- MySQL
- Tailwind CSS
- WebRTC
